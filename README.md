# MRP Drone — Drone Photography Landing Page

Professional landing page for a drone photography and videography business operating in Brazil.

Built as a **freelance project** to establish the client's digital presence — allowing potential customers to discover services, browse aerial work, and reach out to book a session.

**[→ View Live Site](https://grparasky.github.io/mrp-drone-site/)**

---

## Stack

- **React** — component-based UI
- **Vite** — build tooling and dev server
- **CSS Modules** — scoped component styles
- **GitHub Pages** — static hosting with automatic HTTPS

---

## Sections

- **Hero** — full-screen visual with call-to-action
- **About** — introduces the photographer and builds trust
- **Portfolio** — curated showcase of aerial photography and videography
- **Contact** — form and social media links for booking inquiries
- **Social** — links to existing channels to reinforce the brand
